# 👕 Nostra - Fashion Store

A modern and responsive fashion store website built using **HTML**, **CSS**, and **JavaScript**. Nostra provides a clean and user-friendly interface for browsing fashion products across multiple categories.

---

## 📖 Project Overview

Nostra is a front-end fashion e-commerce website developed as a college mini project. It features a modern design, responsive layout, product collections, and a contact page.

---

## ✨ Features

- 🏠 Responsive Home Page
- 🛍️ Fashion Collections Page
- 🔍 Product Search
- 📱 Mobile-Friendly Design
- 📞 Contact Page
- 📧 Newsletter Subscription Section
- 🍔 Responsive Navigation Menu
- 🎨 Clean & Modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser.

No additional software or installation is required.
